
# Kanban Board

website to manage task to do.

## Backend

To deploy this project run

```bash
  cd backend
```
```bash
  npm install
```
```bash
  docker compose up
```

then deploy this in docker

```bash
  npx prisma generate
```
```bash
  npx prisma db push
```

## Frontend

To deploy this project run

```bash
  cd frontend
```
```bash
  npm install
```
```bash
  npm run dev
```
